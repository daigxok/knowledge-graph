# 教师备课工具使用说明

## 🚀 快速开始（3步）

### 第1步：启动本地服务器

**Windows用户**：
```
双击运行：启动教师备课工具.bat
```

**Mac/Linux用户**：
```bash
python3 -m http.server 8000
```

### 第2步：打开浏览器

在浏览器中访问：
```
http://localhost:8000/teacher-quick.html
```

### 第3步：开始使用

1. 从左侧选择章节
2. 点击知识点卡片上的按钮
3. 生成和导出教案

---

## ❓ 为什么需要本地服务器？

**问题**：直接双击HTML文件会出现"加载失败"错误

**原因**：浏览器的CORS安全策略阻止了本地文件访问

**解决**：使用本地服务器（HTTP协议）访问文件

## 📖 详细使用教程

### 功能1：浏览章节

1. 打开页面后，左侧显示所有章节
2. 每个章节后面显示知识点数量
3. 点击章节名称查看该章节的所有知识点

### 功能2：查看知识点

每个知识点卡片包含：
- 知识点名称
- 简短描述
- 两个操作按钮

**按钮说明**：
- **📖 详情** - 查看知识点的详细信息
- **📝 教案** - 立即生成并下载简版教案（1秒完成）

### 功能3：生成教案

#### 方式1：快速教案（推荐）

1. 在章节页面，点击知识点卡片上的"📝 教案"按钮
2. 系统自动生成教案并下载
3. 文件名：`教案-[知识点名称].md`

**特点**：
- ⚡ 1秒完成
- 📄 包含核心内容
- 💾 自动下载

#### 方式2：完整教案

1. 点击知识点卡片上的"📖 详情"按钮
2. 在详情页点击"📝 生成完整教案"按钮
3. 查看完整的教案内容
4. 点击"💾 导出教案"按钮下载

**特点**：
- 📚 内容完整（8个章节）
- 🎯 结构化教案
- 📝 可预览后再导出

### 功能4：教案内容

#### 快速教案包含：
1. 教学目标
2. 知识点概述
3. 教学重点
4. 预计课时

#### 完整教案包含：
1. 教学目标
2. 知识点概述
3. 教学重点
4. 教学方法
5. 教学过程
6. 实际应用
7. 课后作业
8. 教学反思

## 🔧 常见问题

### Q1: 双击HTML文件显示"加载失败"？

**A**: 这是正常的，必须使用本地服务器。

**解决方法**：
1. 双击运行 `启动教师备课工具.bat`
2. 在浏览器中访问 `http://localhost:8000/teacher-quick.html`

### Q2: 提示"python不是内部或外部命令"？

**A**: 系统未安装Python或未添加到环境变量。

**解决方法**：
1. 下载安装Python：https://www.python.org/downloads/
2. 安装时勾选"Add Python to PATH"
3. 重启命令行窗口

### Q3: 端口8000被占用？

**A**: 其他程序正在使用8000端口。

**解决方法**：
修改启动脚本，使用其他端口：
```bash
python -m http.server 8001
```
然后访问：`http://localhost:8001/teacher-quick.html`

### Q4: 教案文件在哪里？

**A**: 教案文件下载到浏览器的默认下载文件夹。

**查找方法**：
- Chrome: 按 Ctrl+J 查看下载内容
- Firefox: 按 Ctrl+Shift+Y 查看下载内容
- Edge: 按 Ctrl+J 查看下载内容

### Q5: 教案格式是什么？

**A**: Markdown格式（.md文件）

**打开方式**：
- 任何文本编辑器（记事本、VS Code等）
- Markdown编辑器（Typora、MarkText等）
- 在线Markdown查看器

**转换方式**：
- 使用Pandoc转换为Word/PDF
- 使用在线转换工具
- 复制粘贴到Word中

### Q6: 可以修改教案内容吗？

**A**: 可以！教案是纯文本文件，可以自由编辑。

**编辑方法**：
1. 用文本编辑器打开.md文件
2. 修改内容
3. 保存文件

### Q7: 如何在其他电脑上使用？

**A**: 复制整个项目文件夹到其他电脑即可。

**步骤**：
1. 复制整个文件夹
2. 在新电脑上安装Python
3. 运行启动脚本
