# 高等数学学域知识图谱系统 - 使用指南

## 📖 快速开始

### 第一步：打开系统

1. **打开浏览器**，推荐使用 Chrome、Edge 或 Firefox
2. **打开 `landing.html` 文件**（首页）
   - 方法1：双击 `landing.html` 文件
   - 方法2：在浏览器中按 `Ctrl+O`，选择 `landing.html`

### 第二步：注册账号

1. 在首页点击 **"立即开始"** 或 **"立即注册"** 按钮
2. 进入注册页面，点击 **"注册"** 标签
3. 填写注册信息：
   - **用户名**：3-20个字符（必填）
   - **密码**：至少6个字符（必填）
   - **确认密码**：再次输入密码（必填）
   - **邮箱**：可选填写
4. 点击 **"注册"** 按钮
5. 注册成功后会自动登录并跳转到知识图谱主界面

### 第三步：开始使用

注册成功后，你会看到知识图谱的主界面，包含：
- **左侧边栏**：搜索、筛选、统计功能
- **中间画布**：知识图谱可视化
- **右上角**：缩放控制按钮
- **顶部**：欢迎信息和退出按钮

## 🎯 主要功能使用

### 1. 浏览知识图谱

**查看节点**：
- 图谱中的每个圆圈代表一个数学知识点
- 不同颜色代表不同的学域（5大学域）
- 节点大小反映重要程度

**查看连接**：
- 节点之间的线条表示知识点的关联
- 实线：前置依赖关系
- 虚线：跨学域关联

**操作方式**：
- **鼠标滚轮**：缩放图谱
- **拖拽空白处**：移动整个图谱
- **点击节点**：查看详细信息
- **悬停节点**：显示快速预览

### 2. 搜索知识点

在左侧边栏顶部的搜索框中：
1. 输入关键词（如"极限"、"导数"、"积分"）
2. 系统会实时筛选匹配的节点
3. 图谱会高亮显示搜索结果

### 3. 按学域筛选

**五大学域**：
- 📈 **变化与逼近**：极限、导数、微分
- 🔄 **结构与累积**：积分、级数
- 🎯 **优化与决策**：极值、优化
- 🎲 **不确定性处理**：级数收敛、概率
- 🌐 **真实问题建模**：应用问题

**筛选方法**：
1. 在左侧边栏找到 **"学域筛选"** 区域
2. 勾选想要查看的学域
3. 图谱会只显示选中学域的节点

### 4. 按难度筛选

在左侧边栏的 **"难度范围"** 区域：
1. 拖动两个滑块设置难度范围（1-5星）
2. 图谱会只显示该难度范围内的节点

### 5. 查看节点详情

**点击任意节点**，右侧会弹出详情面板，显示：
- 节点名称和类型（概念/定理/方法/应用）
- 详细描述
- 所属学域和章节
- 难度等级
- 学习时间估计
- 前置知识和后续知识
- 相关应用案例

### 6. Skills 运用

在节点详情面板中：
1. 点击 **"去运用"** 按钮
2. 查看该知识点的：
   - 核心概念解析
   - 应用场景
   - 练习题
   - 真实案例
   - 学习建议

### 7. 学习路径推荐

1. 点击节点详情面板中的 **"生成学习路径"** 按钮
2. 系统会根据知识依赖关系，推荐最优学习顺序
3. 图谱会高亮显示推荐的学习路径

### 8. 学习进度追踪

在左侧边栏的 **"学习统计"** 区域：
- 查看已完成节点数
- 查看总节点数
- 查看学习进度百分比

**标记完成**：
- 在节点详情面板中点击 **"标记为已完成"**
- 已完成的节点会显示不同的样式

### 9. 导出学习数据

在左侧边栏底部：
1. 点击 **"导出学习数据 JSON"** 按钮
2. 下载包含你学习记录的 JSON 文件
3. 可用于备份或分析

### 10. 多语言切换

系统支持中英文双语：
1. 查找语言切换按钮（通常在右上角）
2. 点击切换到英文或中文
3. 界面会立即更新为选择的语言

## 🔧 高级功能

### 跨学域视图
点击右侧的 **"🔗"** 按钮，查看不同学域之间的关联

### 适应画布
点击右侧的 **"⊡"** 按钮，自动调整视图以显示所有节点

### 重置视图
点击右侧的 **"⟲"** 按钮，恢复到初始视图

### 清除筛选
点击左侧边栏底部的 **"清除所有筛选"** 按钮，重置所有筛选条件

## 🔐 账号管理

### 登录
1. 打开 `landing.html` 或 `auth.html`
2. 输入用户名和密码
3. 可选勾选 **"记住我"**（保持登录状态）

1. 点击右上角的 **"退出"** 按钮
2. 确认退出

- 勾选后，24小时内无需重复登录
- 不勾选，关闭浏览器后需要重新登录

## 💡 使用技巧

### 1. 快速定位知识点
- 使用搜索功能比手动查找更快
- 可以搜索中文名称或关键词

### 2. 系统学习
- 先按学域筛选，系统学习一个学域
- 从低难度到高难度逐步学习
- 利用学习路径推荐功能

### 3. 理解知识关联
- 点击节点查看前置知识和后续知识
- 理解知识点之间的依赖关系
- 按照依赖顺序学习效果更好

### 4. 实践应用
- 每学完一个知识点，查看 Skills 运用
- 完成练习题巩固理解
- 查看真实应用案例

### 5. 定期复习
- 使用学习统计功能追踪进度
- 定期导出学习数据备份
- 重点复习标记为困难的节点

## ❓ 常见问题

### Q1: 忘记密码怎么办？
A: 目前系统是本地存储，如果忘记密码，需要清除浏览器的 localStorage 数据重新注册。

### Q2: 学习数据会丢失吗？
A: 学习数据保存在浏览器本地，建议定期导出备份。清除浏览器数据会导致丢失。

### Q3: 可以在多个设备上使用吗？
A: 可以，但每个设备需要单独注册。学习数据不会自动同步。

### Q4: 图谱显示不完整怎么办？
A: 点击右侧的 **"⊡"** 按钮（适应画布），或使用鼠标滚轮缩小视图。

### Q5: 如何查看所有节点？
A: 点击左侧边栏底部的 **"清除所有筛选"** 按钮，显示所有节点。

### Q6: 节点太多看不清怎么办？
A: 使用学域筛选或难度筛选，减少显示的节点数量。

### Q7: 如何分享给其他人？
A: 将整个项目文件夹复制给其他人，或部署到网站上。

## 🚀 部署到网站

如果想让其他人通过网址访问，可以：

### 方法1：GitHub Pages（免费）
1. 将项目上传到 GitHub
2. 在仓库设置中启用 GitHub Pages
3. 选择主分支作为源
4. 获得网址：`https://你的用户名.github.io/仓库名/landing.html`

### 方法2：Netlify（免费）
1. 访问 netlify.com
2. 拖拽项目文件夹到网站
3. 自动部署并获得网址

详细部署说明请查看 `DEPLOYMENT-GUIDE.md` 文件。
