## 🚀 三步开始

### ⭐ 第一步：打开首页

**双击这个文件** → `landing.html`

或者在浏览器中按 `Ctrl+O`，选择 `landing.html` 文件

---

### ⭐ 第二步：注册账号

1. 在首页点击 **"立即开始"** 按钮
2. 切换到 **"注册"** 标签
3. 填写：
   ```
   用户名：你的名字（3-20个字符）
   密码：你的密码（至少6个字符）
   确认密码：再输入一次密码
   ```
4. 点击 **"注册"** 按钮

✅ 注册成功后会自动登录！

---

### ⭐ 第三步：开始探索

你会看到一个漂亮的知识图谱！

**试试这些操作**：
- 🖱️ 点击任意圆圈（节点）查看详情
- 🔍 在左侧搜索框输入"极限"或"导数"
- 📊 勾选左侧的学域复选框筛选节点
- 🎯 点击节点详情中的"去运用"查看练习

---

## 🎓 学习建议

### 第一次使用
1. 跟随**新手引导**（会自动弹出）
2. 随意点击节点，熟悉界面
3. 尝试搜索和筛选功能

### 系统学习
1. 选择一个学域（如"变化与逼近"）
2. 从低难度（1-2星）开始
3. 点击节点查看详情和前置知识
4. 完成 Skills 练习巩固理解

### 进阶使用
1. 使用学习路径推荐功能
2. 查看跨学域关联
3. 定期导出学习数据备份
