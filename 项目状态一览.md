## 💡 快速开始

### 本地运行
```bash
# 启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx http-server -p 8000

# 访问
http://localhost:8000
```

### 查看空间几何节点
1. 清除浏览器缓存（Ctrl+F5）
2. 搜索"空间"、"向量"、"平面"
3. 或筛选"第12章 空间解析几何"

### 教师备课
1. 选择相关节点
2. 点击"批量操作"
3. 生成教案
4. 导出Markdown/JSON
