═══════════════════════════════════════════════════════════════
                        核心操作流程
═══════════════════════════════════════════════════════════════


1️⃣  搜索知识点
    │
    ├─→ 在左侧搜索框输入关键词
    │   (例如："极限"、"导数"、"积分")
    │
    └─→ 图谱高亮显示匹配结果


2️⃣  筛选节点
    │
    ├─→ 按学域筛选（勾选学域复选框）
    ├─→ 按难度筛选（拖动难度滑块）
    ├─→ 按章节筛选（选择章节下拉框）
    └─→ 按类型筛选（概念/定理/方法/应用）


3️⃣  查看节点详情
    │
    ├─→ 点击图谱中的节点
    │
    ├─→ 右侧弹出详情面板
    │   ┌─────────────────────┐
    │   │ 节点名称            │
    │   │ 详细描述            │
    │   │ 所属学域            │
    │   │ 难度等级            │
    │   │ 前置知识            │
    │   │ 后续知识            │
    │   │ [去运用] [生成路径] │
    │   └─────────────────────┘
    │
    └─→ 点击"去运用"查看练习


4️⃣  Skills 运用
    │
    ├─→ 在详情面板点击"去运用"
    │
    ├─→ 右侧弹出 Skills 面板
    │   ┌─────────────────────┐
    │   │ 🎯 Skill 运用       │
    │   │                     │
    │   │ 核心概念            │
    │   │ 应用场景            │
    │   │ 练习题              │
    │   │ 真实案例            │
    │   │ 学习建议            │
    │   └─────────────────────┘
    │
    └─→ 完成练习，巩固理解


5️⃣  学习路径推荐
    │
    ├─→ 在详情面板点击"生成学习路径"
    │
    ├─→ 系统分析知识依赖关系
    │
    └─→ 图谱高亮显示推荐路径
        (按顺序学习效果更好)


6️⃣  追踪学习进度
    │
    ├─→ 在详情面板标记"已完成"
    │
    ├─→ 左侧统计区域更新进度
    │   ┌─────────────────────┐
    │   │ 已完成: 10          │
    │   │ 总节点: 153         │
    │   │ 进度: 6.5%          │
    │   │ ▓▓░░░░░░░░          │
    │   └─────────────────────┘
    │
    └─→ 点击"导出学习数据"备份


7️⃣  退出登录
    │
    ├─→ 点击右上角"退出"按钮
    │
    ├─→ 确认退出
    │
    └─→ 返回首页 (landing.html)


═══════════════════════════════════════════════════════════════
                        快捷操作
═══════════════════════════════════════════════════════════════

鼠标操作：
  🖱️ 滚轮        → 缩放图谱
  🖱️ 拖拽空白    → 移动图谱
  🖱️ 点击节点    → 查看详情
  🖱️ 悬停节点    → 快速预览

键盘快捷键：
  Ctrl+F       → 聚焦搜索框
  Ctrl+R       → 重置视图
  Esc          → 关闭面板

右侧按钮：
  [+]          → 放大
  [-]          → 缩小
  [⟲]          → 重置视图
  [⊡]          → 适应画布
  [🔗]         → 跨学域视图


═══════════════════════════════════════════════════════════════
                        学习建议流程
═══════════════════════════════════════════════════════════════

新手阶段：
  1. 跟随新手引导（自动弹出）
  2. 随意点击节点，熟悉界面
  3. 尝试搜索和筛选功能
  4. 查看几个节点的详情

入门阶段：
  1. 选择一个学域（如"变化与逼近"）
  2. 筛选低难度节点（1-2星）
  3. 按照前置知识顺序学习
  4. 完成 Skills 练习

进阶阶段：
  1. 使用学习路径推荐
  2. 学习中高难度节点（3-4星）
  3. 查看跨学域关联
  4. 完成综合应用案例

高级阶段：
  1. 挑战高难度节点（5星）
  2. 探索真实问题建模
  3. 导出学习数据分析
  4. 定期复习已学内容


═══════════════════════════════════════════════════════════════
                        常见问题快速解决
═══════════════════════════════════════════════════════════════

Q: 找不到某个知识点？
A: 使用左侧搜索框，输入关键词

Q: 图谱太大看不清？
A: 使用学域筛选或难度筛选，减少显示节点

Q: 不知道从哪学起？
A: 点击节点查看前置知识，从基础开始

Q: 想练习题目？
A: 点击节点详情中的"去运用"按钮

Q: 忘记密码？
A: 清除浏览器数据，重新注册

Q: 学习数据会丢失吗？
A: 定期点击"导出学习数据"备份
