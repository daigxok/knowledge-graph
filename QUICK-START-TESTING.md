# ⚡ 快速开始测试 - 一页指南

## 🚀 启动 (30秒)

```bash
cd knowledge-graph
python -m http.server 8000
```

打开: `http://localhost:8000/test-skills-system.html`

---

## 🎯 6个测试 (20分钟)

### 1️⃣ 梯度可视化 (5分钟)
- 点击"梯度可视化"
- 切换函数类型
- 调整X/Y滑块
- 观察梯度变化

### 2️⃣ 选择题 (3分钟)
- 点击"选择题系统"
- 选择答案
- 查看反馈

### 3️⃣ 计算题 (3分钟)
- 点击"计算题系统"
- 输入答案
- 查看步骤

### 4️⃣ 填空题 (3分钟)
- 点击"填空题系统"
- 填写空格
- 提交验证

### 5️⃣ 判断题 (2分钟)
- 点击"判断题系统"
- 选择正确/错误
- 查看解释

### 6️⃣ 综合测试 (4分钟)
- 点击"综合测试"
- 提交反馈 📝
- 查看统计 📊
- 导出数据 💾
