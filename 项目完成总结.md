## 📊 项目概览

这是一个创新的高等数学学习系统，通过知识图谱可视化展示数学概念之间的关联，帮助学生从传统的章节式学习转向学域式理解。

### 核心理念
**从章节记忆到学域理解 · 5大学域整合范式**

## 🚀 使用方式

### 本地使用（推荐）
1. 下载项目到本地
2. 双击打开 `landing.html`
3. 注册账号并开始使用

### 本地服务器
```bash
# Python
python -m http.server 8000

# Node.js
npx http-server

# 访问 http://localhost:8000/landing.html
```

## ⚠️ 重要说明

### 认证系统
当前为**客户端实现**，适用于：
- ✅ 个人学习使用
- ✅ 演示和教学
- ✅ 本地部署

**不适用于生产环境**，生产环境需要：
- 服务器端认证
- 数据库存储
- 加密传输（HTTPS）
- 安全的密码哈希

### 数据存储
- 数据保存在浏览器 localStorage
- 建议定期导出备份
- 清除浏览器数据会丢失记录
