# 🚀 快速开始测试

## 一键启动

### Windows (PowerShell)
```powershell
cd knowledge-graph
python -m http.server 8000
```

### macOS/Linux (Terminal)
```bash
cd knowledge-graph
python3 -m http.server 8000
```

## 访问测试平台

打开浏览器访问:
```
http://localhost:8000/test-skills-system.html
```

## 测试流程

1. **梯度可视化** → 测试3D可视化
2. **选择题** → 测试答题系统
3. **计算题** → 测试输入验证
4. **填空题** → 测试多空输入
5. **判断题** → 测试快速作答
6. **综合测试** → 提交反馈

## 预计时间

- 完整测试: 20-30分钟
- 快速测试: 10分钟
