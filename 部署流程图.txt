═══════════════════════════════════════════════════════════════
                        常见问题
═══════════════════════════════════════════════════════════════

问题 1: GitHub Desktop 显示很多文件
    │
    └─→ 正常！新增了很多功能，直接提交即可


问题 2: 推送时提示错误
    │
    ├─→ 检查是否已登录 GitHub Desktop
    │
    └─→ 检查网络连接


问题 3: 网站显示 404
    │
    ├─→ 确认 Pages 已启用
    │
    ├─→ 等待 5-10 分钟
    │
    └─→ 清除浏览器缓存 (Ctrl+Shift+Delete)


问题 4: 功能不正常
    │
    ├─→ 按 F12 打开控制台查看错误
    │
    ├─→ 确认所有文件都已上传
    │
    └─→ 清除浏览器缓存
