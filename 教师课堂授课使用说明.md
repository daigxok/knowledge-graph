# 教师课堂授课模式使用说明

## 📌 功能概述

教师课堂授课模式是专为课堂教学设计的简洁界面，让教师能够快速查找、展示和讲解知识点。

## 🚀 快速开始

### 访问方式

1. **从备课中心进入**：`teacher-prep.html` → 点击"课堂授课模式"
2. **直接访问**：打开 `teacher-classroom.html`
3. **从主图进入**：主页右上角 → "教师备课中心" → "课堂授课模式"

## 💡 核心功能

### 1. 章节导航

**左侧边栏显示所有章节**

- 点击章节名称，右侧显示该章节的所有知识点
- 显示每个章节的知识点数量
- 当前选中的章节会高亮显示

**使用场景**：
- 按教学进度快速定位到当前章节
- 查看章节包含的所有知识点
- 规划课堂教学内容

### 2. 快速搜索

**顶部搜索框支持实时搜索**

搜索范围：
- 知识点名称
- 知识点描述
- 关键词

**使用场景**：
- 学生提问时快速查找相关知识点
- 跨章节查找某个概念
- 查找包含特定关键词的所有节点

**示例**：
```
搜索"极限" → 显示所有包含"极限"的知识点
搜索"导数" → 显示所有与导数相关的内容
搜索"应用" → 显示所有应用类节点
```

### 3. 知识点卡片

**每个知识点以卡片形式展示**

卡片信息：
- 📝 知识点名称
- 🎯 难度等级（1-5）
- 📄 简短描述
- 🏷️ 所属学域标签
- 🔘 操作按钮

**两个操作按钮**：
1. **📖 查看详情** - 打开详情弹窗
2. **🖥️ 展示** - 全屏展示（适合投影）

### 4. 详情弹窗

**点击"查看详情"打开详细信息**

包含内容：
- 📝 知识点描述
- 📐 核心公式（LaTeX渲染）
- 🎯 所属学域
- 📚 前置知识
- ⚙️ 学习参数（难度、重要性、学习时间）
- 🏷️ 关键词

**操作**：
- 点击 × 关闭弹窗
- 点击弹窗外部区域关闭
- 支持数学公式实时渲染

### 5. 全屏展示模式

**点击"🖥️ 展示"按钮**

功能：
- 自动进入全屏模式
- 显示知识点详细信息
- 适合课堂投影演示

**使用场景**：
- 课堂讲解时展示知识点
- 投影仪演示
- 大屏幕教学

**退出全屏**：
- 按 ESC 键
- 按 F11 键（部分浏览器）

## 📖 使用流程

### 场景1：按章节授课

```
1. 打开课堂授课页面
2. 左侧点击当前章节（如"第3章 导数与微分"）
3. 右侧显示该章节所有知识点
4. 点击要讲解的知识点卡片
5. 点击"🖥️ 展示"进入全屏模式
6. 开始讲解
```

### 场景2：学生提问快速查找

```
1. 学生提问："老师，什么是极限？"
2. 在搜索框输入"极限"
3. 立即显示所有相关知识点
4. 点击"极限的定义"
5. 点击"🖥️ 展示"全屏讲解
```

### 场景3：复习多个知识点

```
1. 选择要复习的章节
2. 浏览该章节的所有知识点卡片
3. 依次点击"查看详情"
4. 查看前置知识和关联内容
5. 规划复习顺序
```

## ⚡ 快捷操作

### 键盘快捷键

- **ESC** - 关闭弹窗 / 退出全屏
- **F11** - 进入/退出全屏（浏览器原生）
- **Ctrl + F** - 浏览器搜索（可搜索页面内容）

### 鼠标操作

- **单击章节** - 切换章节
- **单击卡片** - 无操作（需点击按钮）
- **单击按钮** - 执行对应操作
- **单击弹窗外** - 关闭弹窗

## 🎯 教学建议

1. 打开课堂授课页面
2. 定位到本节课的章节
3. 浏览要讲解的知识点
4. 检查前置知识是否已讲

**导入环节**：
- 使用搜索功能快速找到前置知识
- 全屏展示复习内容

**新课讲解**：
- 按顺序展示本节知识点
- 利用公式渲染功能讲解数学内容
- 展示学域标签，强调知识点的应用领域

**答疑环节**：
- 学生提问时使用搜索功能
- 快速定位相关知识点
- 展示详细信息回答问题

**总结环节**：
- 浏览本节所有知识点卡片
- 强调重点和难点
- 展示知识点间的关联

- 记录学生常问的问题
- 标记需要重点讲解的知识点
- 规划下节课内容

## 🔧 技术说明

### 浏览器要求

- Chrome 90+（推荐）
- Firefox 88+
- Safari 14+
- Edge 90+

### 功能依赖

- **MathJax 3.2.2** - 数学公式渲染
- **localStorage** - 数据存储
- **ES6 Modules** - 模块化加载

### 数据来源

- 节点数据：`data/nodes.json`
- 学域数据：`data/domains.json`
- 通过 `NodeDataManager` 模块加载

## ❓ 常见问题

### Q1: 为什么看不到某些章节？

**A**: 可能原因：
- 数据尚未加载完成（等待几秒）
- 该章节没有知识点
- 浏览器缓存问题（刷新页面）

### Q2: 数学公式显示不正常？

**A**: 解决方法：
- 检查网络连接（MathJax需要在线加载）
- 等待几秒让MathJax完成渲染
- 刷新页面重新加载

### Q3: 全屏模式无法退出？

**A**: 退出方法：
- 按 ESC 键
- 按 F11 键
- 点击浏览器地址栏
- 移动鼠标到屏幕顶部（部分浏览器会显示退出按钮）

### Q4: 搜索没有结果？

**A**: 检查：
- 搜索词是否正确
- 尝试使用更简短的关键词
- 尝试搜索知识点的部分名称

### Q5: 卡片显示不完整？

**A**: 解决方法：
- 调整浏览器窗口大小
- 使用浏览器缩放功能（Ctrl + 滚轮）
- 刷新页面

## 🎨 界面特点

### 设计理念

- **简洁** - 去除复杂功能，专注课堂教学
- **快速** - 快速查找和展示知识点
- **直观** - 卡片式布局，一目了然

- **主色调** - 紫色渐变（#667eea → #764ba2）
