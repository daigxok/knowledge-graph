## 二、详细知识节点列表

### 模块1：空间向量代数（Vector Algebra）

#### 1.1 空间向量基础
**节点ID**: spatial-vector-basics

**子节点**：
1. **空间直角坐标系**
   - ID: spatial-coordinate-system
   - 描述：三维空间中的坐标系建立
   - 前置知识：平面直角坐标系
   - 关键概念：坐标轴、坐标平面、卦限

2. **空间向量的概念**
   - ID: spatial-vector-concept
   - 描述：空间中的有向线段
   - 前置知识：平面向量
   - 关键概念：模、方向、单位向量

3. **空间向量的坐标表示**
   - ID: spatial-vector-coordinates
   - 描述：用坐标表示空间向量
   - 前置知识：空间直角坐标系
   - 关键概念：坐标分量、位置向量

4. **向量的模与方向余弦**
   - ID: vector-magnitude-direction
   - 描述：向量的长度和方向角
   - 前置知识：空间向量坐标
   - 关键概念：模长公式、方向余弦、方向角

#### 1.2 空间向量的运算
**节点ID**: spatial-vector-operations

**子节点**：
5. **向量的线性运算**
   - ID: vector-linear-operations
   - 描述：向量的加法、减法、数乘
   - 前置知识：平面向量运算
   - 关键概念：平行四边形法则、三角形法则

6. **向量的数量积（点积）**
   - ID: vector-dot-product
   - 描述：两向量的数量积运算
   - 前置知识：平面向量数量积
   - 关键概念：$\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos\theta$
   - 应用：计算夹角、判断垂直

7. **向量的向量积（叉积）**
   - ID: vector-cross-product
   - 描述：两向量的向量积运算
   - 前置知识：向量坐标表示
   - 关键概念：$\vec{a} \times \vec{b}$、右手法则
   - 应用：求法向量、计算面积

8. **向量的混合积**
   - ID: vector-mixed-product
   - 描述：三个向量的混合积
   - 前置知识：数量积、向量积
   - 关键概念：$(\vec{a} \times \vec{b}) \cdot \vec{c}$
   - 应用：判断共面、计算体积

#### 1.3 空间向量的应用
**节点ID**: spatial-vector-applications

**子节点**：
9. **两点间距离公式**
   - ID: distance-between-points
   - 描述：空间中两点间的距离
   - 前置知识：向量的模
   - 公式：$d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2}$

10. **向量的投影**
    - ID: vector-projection
    - 描述：一个向量在另一个向量上的投影
    - 前置知识：数量积
    - 公式：$\text{proj}_{\vec{b}}\vec{a} = \frac{\vec{a} \cdot \vec{b}}{|\vec{b}|}$

11. **向量的夹角**
    - ID: angle-between-vectors
    - 描述：两向量之间的夹角
    - 前置知识：数量积
    - 公式：$\cos\theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}||\vec{b}|}$

### 模块2：空间平面与直线（Planes and Lines）

#### 2.1 平面方程
**节点ID**: plane-equations

**子节点**：
12. **平面的点法式方程**
    - ID: plane-point-normal-form
    - 描述：已知点和法向量的平面方程
    - 前置知识：向量的数量积
    - 公式：$A(x-x_0) + B(y-y_0) + C(z-z_0) = 0$

13. **平面的一般方程**
    - ID: plane-general-form
    - 描述：平面的标准形式
    - 前置知识：点法式方程
    - 公式：$Ax + By + Cz + D = 0$

14. **平面的截距式方程**
    - ID: plane-intercept-form
    - 描述：已知三个截距的平面方程
    - 前置知识：一般方程
    - 公式：$\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$

15. **平面束方程**
    - ID: plane-pencil
    - 描述：过两平面交线的平面族
    - 前置知识：平面一般方程
    - 公式：$A_1x + B_1y + C_1z + D_1 + \lambda(A_2x + B_2y + C_2z + D_2) = 0$

#### 2.2 空间直线方程
**节点ID**: line-equations

**子节点**：
16. **直线的参数方程**
    - ID: line-parametric-form
    - 描述：用参数表示的直线方程
    - 前置知识：向量的线性运算
    - 公式：$\begin{cases} x = x_0 + mt \\ y = y_0 + nt \\ z = z_0 + pt \end{cases}$

17. **直线的对称式方程**
    - ID: line-symmetric-form
    - 描述：直线的标准形式
    - 前置知识：参数方程
    - 公式：$\frac{x-x_0}{m} = \frac{y-y_0}{n} = \frac{z-z_0}{p}$

18. **直线的一般方程**
    - ID: line-general-form
    - 描述：两平面交线表示的直线
    - 前置知识：平面方程
    - 公式：$\begin{cases} A_1x + B_1y + C_1z + D_1 = 0 \\ A_2x + B_2y + C_2z + D_2 = 0 \end{cases}$

19. **直线的方向向量**
    - ID: line-direction-vector
    - 描述：表示直线方向的向量
    - 前置知识：向量概念
    - 关键概念：方向数、方向余弦

#### 2.3 平面与直线的位置关系
**节点ID**: plane-line-relations

**子节点**：
20. **两平面的位置关系**
    - ID: two-planes-relation
    - 描述：平行、相交、重合
    - 前置知识：平面方程、向量积
    - 判断方法：法向量的关系

21. **两直线的位置关系**
    - ID: two-lines-relation
    - 描述：平行、相交、异面、重合
    - 前置知识：直线方程、混合积
    - 判断方法：方向向量和连接向量

22. **直线与平面的位置关系**
    - ID: line-plane-relation
    - 描述：平行、相交、直线在平面内
    - 前置知识：直线方程、平面方程
    - 判断方法：方向向量与法向量

23. **两平面的夹角**
    - ID: angle-between-planes
    - 描述：两平面法向量的夹角
    - 前置知识：向量夹角
    - 公式：$\cos\theta = \frac{|A_1A_2 + B_1B_2 + C_1C_2|}{\sqrt{A_1^2+B_1^2+C_1^2}\sqrt{A_2^2+B_2^2+C_2^2}}$

24. **直线与平面的夹角**
    - ID: angle-line-plane
    - 描述：直线与平面的夹角
    - 前置知识：向量夹角
    - 公式：$\sin\theta = \frac{|Am + Bn + Cp|}{\sqrt{A^2+B^2+C^2}\sqrt{m^2+n^2+p^2}}$

#### 2.4 距离计算
**节点ID**: distance-calculations

**子节点**：
25. **点到平面的距离**
    - ID: point-to-plane-distance
    - 描述：空间中点到平面的距离
    - 前置知识：平面方程、向量投影
    - 公式：$d = \frac{|Ax_0 + By_0 + Cz_0 + D|}{\sqrt{A^2+B^2+C^2}}$

26. **点到直线的距离**
    - ID: point-to-line-distance
    - 描述：空间中点到直线的距离
    - 前置知识：向量积
    - 公式：$d = \frac{|\vec{PM} \times \vec{s}|}{|\vec{s}|}$

27. **两平行平面的距离**
    - ID: parallel-planes-distance
    - 描述：两平行平面之间的距离
    - 前置知识：点到平面距离
    - 方法：转化为点到平面距离

28. **两异面直线的距离**
    - ID: skew-lines-distance
    - 描述：两条异面直线之间的距离
    - 前置知识：混合积
    - 公式：$d = \frac{|(\vec{P_1P_2}, \vec{s_1}, \vec{s_2})|}{|\vec{s_1} \times \vec{s_2}|}$

### 模块3：空间曲面与曲线（Surfaces and Curves）

#### 3.1 空间曲面
**节点ID**: spatial-surfaces

**子节点**：
29. **曲面的方程**
    - ID: surface-equation
    - 描述：空间曲面的一般方程
    - 前置知识：三元函数
    - 形式：$F(x, y, z) = 0$

30. **旋转曲面**
    - ID: surface-of-revolution
    - 描述：曲线绕坐标轴旋转形成的曲面
    - 前置知识：曲面方程
    - 类型：绕x轴、y轴、z轴旋转

31. **柱面**
    - ID: cylindrical-surface
    - 描述：平行于某坐标轴的直线沿曲线移动形成的曲面
    - 前置知识：曲面方程
    - 特点：方程中缺少一个变量

32. **锥面**
    - ID: conical-surface
    - 描述：过定点的直线沿曲线移动形成的曲面
    - 前置知识：曲面方程
    - 特点：方程是齐次的

33. **球面**
    - ID: sphere
    - 描述：到定点距离为定值的点的轨迹
    - 前置知识：距离公式
    - 方程：$(x-a)^2 + (y-b)^2 + (z-c)^2 = R^2$

#### 3.2 二次曲面
**节点ID**: quadric-surfaces

**子节点**：
34. **椭球面**
    - ID: ellipsoid
    - 描述：三个方向都有界的二次曲面
    - 方程：$\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1$
    - 特点：封闭曲面

35. **单叶双曲面**
    - ID: hyperboloid-one-sheet
    - 描述：一个方向无界的二次曲面
    - 方程：$\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$
    - 特点：连通曲面

36. **双叶双曲面**
    - ID: hyperboloid-two-sheets
    - 描述：两个方向无界的二次曲面
    - 方程：$\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = -1$
    - 特点：两个分离的曲面

37. **椭圆抛物面**
    - ID: elliptic-paraboloid
    - 描述：开口向上或向下的抛物面
    - 方程：$z = \frac{x^2}{a^2} + \frac{y^2}{b^2}$
    - 特点：碗状曲面

38. **双曲抛物面（马鞍面）**
    - ID: hyperbolic-paraboloid
    - 描述：马鞍形状的抛物面
    - 方程：$z = \frac{x^2}{a^2} - \frac{y^2}{b^2}$
    - 特点：鞍点曲面

39. **椭圆锥面**
    - ID: elliptic-cone
    - 描述：锥形的二次曲面
    - 方程：$\frac{x^2}{a^2} + \frac{y^2}{b^2} = \frac{z^2}{c^2}$
    - 特点：顶点在原点

40. **椭圆柱面**
    - ID: elliptic-cylinder
    - 描述：椭圆沿z轴平移形成的柱面
    - 方程：$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$
    - 特点：方程中缺z

41. **双曲柱面**
    - ID: hyperbolic-cylinder
    - 描述：双曲线沿z轴平移形成的柱面
    - 方程：$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$
    - 特点：方程中缺z

42. **抛物柱面**
    - ID: parabolic-cylinder
    - 描述：抛物线沿z轴平移形成的柱面
    - 方程：$y^2 = 2px$
    - 特点：方程中缺z

#### 3.3 空间曲线
**节点ID**: spatial-curves

**子节点**：
43. **空间曲线的一般方程**
    - ID: curve-general-form
    - 描述：两曲面交线表示的曲线
    - 前置知识：曲面方程
    - 形式：$\begin{cases} F(x,y,z) = 0 \\ G(x,y,z) = 0 \end{cases}$

44. **空间曲线的参数方程**
    - ID: curve-parametric-form
    - 描述：用参数表示的曲线方程
    - 前置知识：参数方程
    - 形式：$\begin{cases} x = x(t) \\ y = y(t) \\ z = z(t) \end{cases}$

45. **空间曲线在坐标面上的投影**
    - ID: curve-projection
    - 描述：空间曲线在三个坐标面上的投影
    - 前置知识：曲线方程
    - 方法：消去一个变量

46. **螺旋线**
    - ID: helix
    - 描述：圆柱螺旋线
    - 前置知识：参数方程
    - 方程：$\begin{cases} x = a\cos t \\ y = a\sin t \\ z = bt \end{cases}$
