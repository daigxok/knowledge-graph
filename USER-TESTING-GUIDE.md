## 📝 测试任务

### 任务1: 梯度可视化测试 (5分钟)

**目标**: 测试3D可视化的交互性和直观性

**步骤**:
1. 点击"梯度可视化"卡片
2. 观察初始显示的抛物面和梯度向量
3. 切换函数类型到"鞍面"
4. 调整X坐标滑块，观察梯度变化
5. 调整Y坐标滑块，观察梯度变化
6. 取消勾选"显示等高线"
7. 取消勾选"显示梯度向量"
8. 点击"重置视图"

**评估要点**:
- [ ] 可视化是否清晰易懂？
- [ ] 控制是否响应迅速？
- [ ] 梯度信息是否准确显示？
- [ ] 是否帮助理解梯度概念？

**反馈**: 在测试后点击"提交反馈"按钮

---

### 任务2: 选择题系统测试 (3分钟)

**目标**: 测试选择题的显示和反馈机制

**步骤**:
1. 点击"选择题系统"卡片
2. 阅读题目
3. 故意选择错误答案并提交
4. 观察错误反馈
5. 再次选择错误答案
6. 观察是否显示正确答案
7. 点击"提示"按钮
8. 选择正确答案并提交

**评估要点**:
- [ ] 题目是否清晰？
- [ ] 反馈是否及时？
- [ ] 解释是否有帮助？
- [ ] 提示功能是否有用？

---

### 任务3: 计算题系统测试 (3分钟)

**目标**: 测试计算题的输入和验证

**步骤**:
1. 点击"计算题系统"卡片
2. 阅读题目
3. 输入错误答案"0"并提交
4. 观察反馈
5. 点击"查看步骤"
6. 输入正确答案"1"并提交
7. 观察成功反馈

**评估要点**:
- [ ] 输入框是否易用？
- [ ] 答案验证是否准确？
- [ ] 步骤提示是否清晰？
- [ ] 反馈是否有帮助？

---

### 任务4: 填空题系统测试 (3分钟)

**目标**: 测试多空输入和验证

**步骤**:
1. 点击"填空题系统"卡片
2. 在第一个空填入"0"
3. 在第二个空填入"h"
4. 提交答案
5. 观察反馈

**评估要点**:
- [ ] 填空位置是否明显？
- [ ] 多空输入是否方便？
- [ ] 验证是否准确？
- [ ] 整体体验如何？

---

### 任务5: 判断题系统测试 (2分钟)

**目标**: 测试判断题的快速作答

**步骤**:
1. 点击"判断题系统"卡片
2. 选择"正确"
3. 提交答案
4. 观察反馈和解释

**评估要点**:
- [ ] 选项是否清晰？
- [ ] 作答是否快速？
- [ ] 反馈是否准确？
- [ ] 解释是否充分？

---

### 任务6: 综合测试 (5分钟)

**目标**: 测试整体集成和用户流程

**步骤**:
1. 点击"综合测试"卡片
2. 点击"📝 提交反馈"
3. 给出评分（点击星星）
4. 勾选有帮助的功能
5. 填写建议或意见
6. 选择是否推荐
7. 提交反馈
8. 点击"📊 查看统计"
9. 查看反馈统计数据
10. 点击"💾 导出数据"

**评估要点**:
- [ ] 反馈表单是否易用？
- [ ] 统计显示是否清晰？
- [ ] 数据导出是否成功？
- [ ] 整体流程是否流畅？

## 🎯 评估标准

### 可用性 (1-5分)

| 分数 | 标准 |
|------|------|
| 5 | 非常容易使用，无需说明 |
| 4 | 容易使用，偶尔需要思考 |
| 3 | 可以使用，需要一些学习 |
| 2 | 较难使用，经常困惑 |
| 1 | 很难使用，无法完成任务 |

### 功能性 (1-5分)

| 分数 | 标准 |
|------|------|
| 5 | 功能完整，完全满足需求 |
| 4 | 功能较完整，基本满足需求 |
| 3 | 功能一般，部分满足需求 |
| 2 | 功能不足，难以满足需求 |
| 1 | 功能缺失，无法满足需求 |

### 用户体验 (1-5分)

| 分数 | 标准 |
|------|------|
| 5 | 非常愉悦，超出预期 |
| 4 | 愉悦，符合预期 |
| 3 | 一般，基本可接受 |
| 2 | 不佳，有些失望 |
| 1 | 很差，非常失望 |

### 性能 (1-5分)

| 分数 | 标准 |
|------|------|
| 5 | 非常快速，无延迟 |
| 4 | 快速，偶尔延迟 |
| 3 | 一般，有时延迟 |
| 2 | 较慢，经常延迟 |
| 1 | 很慢，严重延迟 |

## 🐛 常见问题

### Q: 可视化不显示怎么办？
A: 刷新页面，确保浏览器支持Canvas

### Q: 答案提交没反应？
A: 检查是否填写了所有必填项

### Q: 统计数据不准确？
A: 清除浏览器缓存后重试

### Q: 如何重置测试？
A: 刷新页面即可重新开始
