# 🚀 快速入门指南

## 三步开始使用

### 1️⃣ 打开系统
双击 **`landing.html`** 文件，在浏览器中打开首页

### 2️⃣ 注册账号
1. 点击 **"立即开始"** 按钮
2. 切换到 **"注册"** 标签
3. 填写用户名（3-20字符）和密码（至少6字符）
4. 点击 **"注册"** 按钮

### 3️⃣ 开始探索
注册成功后自动进入知识图谱界面！

---

## 🎯 核心操作

| 操作 | 方法 |
|------|------|
| **查看节点详情** | 点击图谱中的圆圈 |
| **搜索知识点** | 左侧搜索框输入关键词 |
| **缩放图谱** | 鼠标滚轮 或 右侧 +/- 按钮 |
| **移动图谱** | 拖拽空白区域 |
| **筛选学域** | 左侧勾选学域复选框 |
| **查看练习** | 节点详情 → 点击"去运用" |
| **退出登录** | 右上角"退出"按钮 |

## 📊 五大学域

- 📈 **变化与逼近** - 极限、导数、微分
- 🔄 **结构与累积** - 积分、级数
- 🎯 **优化与决策** - 极值、优化
- 🎲 **不确定性处理** - 级数收敛
- 🌐 **真实问题建模** - 应用问题

---

## 💡 学习建议

1. **跟随新手引导**（首次使用自动显示）
2. **选择一个学域**，从低难度开始
3. **点击节点**查看详情和前置知识
4. **完成练习**巩固理解
5. **定期导出**学习数据备份

---

## ⚠️ 注意事项

- ✅ 学习数据保存在浏览器本地
- ✅ 建议定期导出数据备份
- ✅ 支持手机和平板访问
- ⚠️ 清除浏览器数据会丢失学习记录
- ⚠️ 忘记密码需要重新注册
