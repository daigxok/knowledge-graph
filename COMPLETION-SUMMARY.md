## 🚀 部署和运行

### 快速启动
```bash
cd knowledge-graph
python -m http.server 8000
# 访问: http://localhost:8000
```

### 系统要求
- 浏览器: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- 网络: 需要访问CDN (D3.js, MathJax)
- 存储: 需要localStorage支持

### 测试验证
```bash
# 访问测试页面
```
