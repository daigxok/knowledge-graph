# 教师备课简明指南

## 🎯 3分钟快速备课

### 第一步：选择知识点（30秒）
1. 打开知识图谱页面
2. 使用搜索框或章节筛选找到目标知识点
3. 按住Ctrl键，点击多个相关节点

### 第二步：生成教案（1分钟）
1. 点击右下角"批量操作"按钮
2. 选择"生成教案"
3. 系统自动生成结构化教案

### 第三步：导出使用（30秒）
1. 点击"导出教案"
2. 选择Markdown格式（推荐）
3. 保存到本地，即可使用

## 📚 常用功能速查

### 快速搜索
```
搜索框输入：
- 知识点名称（如"导数"）
- 章节号（如"第3章"）
- 应用场景（如"优化"）
```

### 快速筛选
```
章节筛选：选择具体章节
学域筛选：选择学域类型
难度筛选：选择难度等级
```

### 快速选择
```
单选：点击节点
多选：Ctrl + 点击
全选：使用批量操作面板
```

## 🎓 典型备课场景

### 场景1：备一节新课（5分钟）
1. 搜索本节课主题（如"极限"）
2. 选择3-5个核心节点
3. 生成教案
4. 查看前置知识和应用案例
5. 导出教案

### 场景2：复习课备课（3分钟）
1. 选择章节筛选
2. 批量选择该章节所有节点
3. 生成综合教案
4. 导出使用

### 场景3：专题课备课（8分钟）
1. 搜索专题关键词
2. 选择跨章节相关节点
3. 查看知识关联
4. 生成专题教案
5. 添加自定义内容
6. 导出使用

## 💡 备课技巧

### 技巧1：利用前置知识
- 查看节点的前置知识
- 确保学生已掌握基础
- 设计复习环节

### 技巧2：关联工程案例
- 每个节点都有应用案例
- 选择贴近学生的案例
- 增强学习动机

### 技巧3：使用GeoGebra
- 点击节点的GeoGebra链接
- 课堂演示动态效果
- 增强直观理解

### 技巧4：导出多种格式
- Markdown：便于编辑
- JSON：便于程序处理
- 可转换为Word/PDF

## ⚡ 快捷键

```
Ctrl + 点击：多选节点
Ctrl + A：全选（在批量操作面板中）
Ctrl + F：搜索
Esc：取消选择
```

## 📋 教案模板

### 标准教案包含
1. **课程信息**
   - 课程名称
   - 知识点列表
   - 学时安排

2. **教学目标**
   - 知识目标
   - 能力目标
   - 应用目标

3. **教学内容**
   - 核心概念
   - 前置知识
   - 知识关联

4. **教学方法**
   - 讲授法
   - 演示法（GeoGebra）
   - 案例法

5. **工程应用**
   - 实际案例
   - 应用场景
   - 拓展阅读

## 🔧 常见问题

### Q1：如何快速找到相关知识点？
**A**：使用搜索功能，输入关键词即可

### Q2：如何选择多个节点？
**A**：按住Ctrl键，依次点击需要的节点

### Q3：教案可以修改吗？
**A**：可以，导出为Markdown后可自由编辑

### Q4：如何保存备课记录？
**A**：导出的教案文件即为备课记录

### Q5：可以分享给其他老师吗？
**A**：可以，导出的文件可以直接分享
