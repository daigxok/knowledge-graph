## 🚀 快速开始 (3步)

### 步骤1: 启动服务器

```bash
# 进入项目目录
cd knowledge-graph

# 启动HTTP服务器
python -m http.server 8000
```

**看到这个输出就成功了**:
```
Serving HTTP on :: port 8000
```

### 步骤2: 打开浏览器

在浏览器中访问:
```
http://localhost:8000
```

### 步骤3: 开始使用

等待2-3秒，加载指示器消失后，你就可以开始探索知识图谱了！

## 🎯 主要功能使用

### 1. 浏览知识图谱

#### 查看整体结构
- 页面加载后，你会看到一个交互式的知识图谱
- **节点** (圆圈) 代表知识点
- **连线** 代表知识点之间的关系

#### 缩放和平移
- **放大**: 点击右上角的 `+` 按钮
- **缩小**: 点击右上角的 `-` 按钮
- **重置视图**: 点击 `⟲` 按钮
- **鼠标操作**: 
  - 滚轮缩放
  - 拖拽平移

#### 节点交互
- **点击节点**: 查看详细信息
- **拖拽节点**: 调整节点位置
- **悬停节点**: 显示快速信息

---

### 2. 使用侧边栏筛选

#### 搜索知识点
1. 在左侧侧边栏顶部找到搜索框
2. 输入关键词 (如 "极限"、"导数"、"积分")
3. 系统会自动高亮匹配的节点

#### 按学域筛选
1. 在侧边栏找到 "学域筛选" 部分
2. 勾选你感兴趣的学域:
   - 📊 变化与逼近
   - 📈 结构与累积
   - 🎯 优化与决策
   - 🔄 不确定性处理
   - 🌍 真实问题建模
3. 图谱会只显示选中学域的节点

#### 按章节筛选
1. 找到 "传统章节" 下拉菜单
2. 选择特定章节
3. 图谱会显示该章节的知识点

#### 按难度筛选
1. 找到 "难度范围" 滑块
2. 调整最小和最大难度
3. 图谱会显示符合难度范围的节点

#### 清除筛选
点击侧边栏底部的 "清除所有筛选" 按钮

---

### 3. 查看节点详情

#### 打开详情面板
1. 点击任意节点
2. 右侧会滑出详情面板

#### 详情面板内容
- **节点名称**: 知识点的名称
- **所属学域**: 该知识点属于哪个学域
- **难度等级**: 1-5星
- **描述**: 知识点的详细说明
- **公式**: 相关的数学公式 (如果有)
- **前置知识**: 需要先掌握的知识点
- **相关Skills**: 可用的学习工具

#### 关闭详情面板
点击详情面板右上角的 `×` 按钮

---

### 4. 使用学习路径

#### 生成学习路径
1. 打开节点详情面板
2. 找到 "学习路径" 部分
3. 点击 "生成学习路径" 按钮
4. 系统会自动生成从基础到该节点的学习路径

#### 查看路径步骤
- 每个步骤显示:
  - 步骤编号
  - 知识点名称
  - 学习原因
  - 预计时间
  - 难度等级

#### 路径可视化
- 图谱上会高亮显示路径
- 路径节点用绿色标记
- 路径连线用绿色加粗显示
- 每个节点显示步骤编号

#### 清除路径
点击 "清除路径" 按钮

---

### 5. 使用Skills系统

#### 什么是Skills?
Skills是增强学习体验的交互式工具，包括:
- 🎨 可视化工具
- 🎬 动画演示
- 🎮 交互练习
- 📝 应用案例

#### 查看可用Skills
1. 打开节点详情面板
2. 滚动到 "相关Skills" 部分
3. 你会看到该节点可用的Skills列表

#### 激活Skill
1. 点击Skill按钮 (如 "🎨 梯度可视化Skill")
2. 右侧会打开Skill面板
3. 面板包含4个标签页:
   - **理论**: 概念讲解
   - **可视化**: 交互式图形
   - **练习**: 练习题
   - **应用**: 实际应用案例

#### 使用Skill内容

**理论标签页**:
- 阅读概念讲解
- 查看数学公式
- 理解核心概念

**可视化标签页**:
- 点击 "启动可视化" 按钮
- 与3D图形交互
- 调整参数观察变化

**练习标签页**:
- 按难度分组 (基础/中级/高级)
- 点击 "查看" 按钮做题
- 获得即时反馈

**应用标签页**:
- 查看实际应用案例
- 了解知识点的实用价值

#### 关闭Skill面板
点击Skill面板右上角的 `×` 按钮

---

### 6. 跨学域视图

#### 什么是跨学域视图?
显示不同学域之间的知识点连接，帮助你理解知识的整体结构。

#### 启用跨学域视图
1. 点击右上角的 `🔗` 按钮
2. 图谱会只显示跨学域的连接
3. 虚线表示跨学域关系

#### 退出跨学域视图
再次点击 `🔗` 按钮

## 🔧 常见问题

### Q1: 加载指示器不消失怎么办?

**解决方法**:
1. 等待5秒
2. 如果还不消失，按 `Ctrl+F5` 硬刷新
3. 检查浏览器控制台 (F12) 是否有错误
4. 确保服务器正在运行

### Q2: 节点太多看不清怎么办?

**解决方法**:
1. 使用筛选功能减少显示的节点
2. 使用缩放功能放大查看
3. 使用搜索功能定位特定节点

### Q3: 如何保存学习进度?

**解决方法**:
系统会自动保存你的:
- 筛选设置
- 视图位置
- 学习路径

下次打开会自动恢复。

### Q4: Skills面板打不开怎么办?

**解决方法**:
1. 确保点击的是Skill按钮
2. 检查浏览器控制台是否有错误
3. 刷新页面重试

### Q5: 如何查看所有Skills?

**解决方法**:
1. 浏览不同的节点
2. 每个节点显示其相关的Skills
3. 或查看文档 `SKILLS-ARCHITECTURE.md`

## 📱 移动端使用

### 触摸操作
- **单指拖动**: 平移图谱
- **双指捏合**: 缩放图谱
- **点击节点**: 查看详情
- **侧边栏**: 点击 `☰` 按钮打开/关闭

### 注意事项
- 移动端屏幕较小，建议使用筛选功能
- Skill面板会全屏显示
- 某些复杂可视化可能性能较低
