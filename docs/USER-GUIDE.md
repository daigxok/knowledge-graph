## 🚀 快速开始

### 首次访问

1. **打开系统**
   - 在浏览器中打开 `index.html`
   - 系统会自动加载知识图谱

2. **选择语言**
   - 点击右上角的语言切换器 🌐
   - 选择中文或English
   - 系统会记住您的选择

3. **浏览图谱**
   - 使用鼠标拖动平移视图
   - 滚轮缩放图谱
   - 点击节点查看详情

## 📚 基础功能

### 1. 浏览知识图谱

#### 查看节点
- **点击节点**: 在右侧面板显示详细信息
- **悬停节点**: 显示节点名称和基本信息
- **双击节点**: 居中显示该节点

#### 节点信息
每个节点包含：
- 📝 名称（中英文）
- 📖 详细描述
- 🎯 难度等级（1-5星）
- ⏱️ 预计学习时间
- 🔗 前置知识
- 🎯 相关技能
- 💼 实际应用

#### 视图控制
- **放大**: 点击 `+` 按钮或滚轮向上
- **缩小**: 点击 `-` 按钮或滚轮向下
- **重置**: 点击 `⟲` 按钮恢复初始视图
- **适应**: 点击 `⊡` 按钮自动适应画布
- **跨域视图**: 点击 `🔗` 按钮查看跨学域关系

---

### 2. 搜索和过滤

#### 关键词搜索
1. 在左侧搜索框输入关键词
2. 系统实时显示匹配结果
3. 点击结果跳转到对应节点

**搜索范围**:
- 节点名称（中英文）
- 节点描述
- 关键词标签

#### 学域筛选
选择一个或多个学域：
- 📈 变化与逼近
- 🔄 结构与累积
- 🎯 优化与决策
- 🎲 不确定性处理
- 🌐 真实问题建模

#### 难度筛选
拖动滑块选择难度范围：
- 1星: 入门
- 2星: 基础
- 3星: 中级
- 4星: 高级
- 5星: 专家

#### 节点类型筛选
- ☑️ 概念
- ☑️ 定理
- ☑️ 方法
- ☑️ 应用

#### 清除筛选
点击"清除所有筛选"按钮恢复显示所有节点

### 3. 学习路径

#### 查看推荐路径
1. 点击目标节点
2. 系统自动显示前置知识
3. 按顺序学习推荐节点

#### 自定义路径
1. 选择起始节点
2. 选择目标节点
3. 系统计算最短学习路径
4. 显示预计学习时间

#### 进度追踪
- 标记已完成的节点
- 查看学习进度百分比
- 查看剩余学习时间

---

### 4. 查看应用案例

#### 浏览案例
1. 点击节点查看详情
2. 滚动到"实际应用"部分
3. 点击案例标题展开详情

#### 案例内容
每个案例包含：
- 🎯 问题背景
- 📐 数学建模
- 🔧 求解方法
- 💻 代码实现
- 📊 可视化图表
- 💡 实际影响

#### 运行代码
1. 复制代码到本地环境
2. 安装所需依赖
3. 运行查看结果

### 5. Skills 运用

#### 进入 Skills
1. 点击左侧"Skills 运用"
2. 选择相关节点
3. 点击"去运用"按钮

#### Skills 内容

1. 阅读题目
2. 尝试解答
3. 查看提示（如需要）
4. 对照详细解答
5. 标记完成状态

### 2. 导出功能

#### 导出为 PDF
1. 点击导出按钮
2. 选择"导出为 PDF"
3. 选择要导出的内容
4. 点击"生成 PDF"
5. 下载文件

**可导出内容**:
- 当前学习路径
- 选定节点详情
- 完整知识图谱

#### 导出为 Markdown
1. 选择要导出的节点
2. 点击"导出为 Markdown"
3. 下载 .md 文件

**用途**:
- 制作学习笔记
- 编写文档
- 分享知识

#### 导出图谱截图
1. 调整图谱视图
2. 点击"导出为图片"
3. 下载 PNG 文件

**用途**:
- 制作演示文稿
- 撰写论文
- 教学展示

#### 导出学习进度
1. 点击"导出数据"
2. 下载 JSON 文件
3. 保存学习进度

**用途**:
- 备份进度
- 跨设备同步
- 数据分析

## ❓ 常见问题

### Q1: 如何开始学习？
**A**: 建议从 Phase 1 基础节点开始，按学域理解知识结构，利用学习路径功能系统学习。

### Q2: 可视化无法显示？
**A**: 确保浏览器支持 WebGL，更新到最新版本，检查网络连接是否正常。

### Q3: 如何保存学习进度？
**A**: 系统自动保存到浏览器本地存储，也可以导出 JSON 文件备份。

### Q4: 移动端性能慢？
**A**: 尝试减少显示的节点数量，使用过滤功能，关闭不必要的可视化。

### Q5: 如何分享给他人？
**A**: 使用分享功能生成链接，或导出 PDF/Markdown 文件。

### Q6: 支持离线使用吗？
**A**: 首次访问后会缓存资源，可以离线浏览已加载的内容。

### Q7: 如何反馈问题？
**A**: 查看 FAQ 文档或联系技术支持。

2. 查看相关应用案例
3. 学习所需数学知识
