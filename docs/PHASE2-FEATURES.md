# Phase 2 功能介绍

**版本**: 2.0.0  
**发布日期**: 2026-02-27  
**文档类型**: 功能介绍

---

## 📋 概述

Phase 2 内容深度扩展在 Phase 1 的 50 个基础节点基础上，新增 75 个高级数学概念节点，深化 Skills 模块内容，完善 100 个行业应用案例，并建立完整的节点关系网络，使系统从基础教学工具发展为包含深度内容和实际应用的完整学习平台。

---

## 🎯 核心特性

### 1. 扩展的知识图谱 (150个节点)

**Phase 1 基础** (50个节点):
- 函数极限与连续
- 导数与微分
- 积分概念
- 多元函数基础

**Phase 2 扩展** (75个节点):
- Domain-1: 20个高级主题节点（曲率、函数作图、不等式证明）
- Domain-2: 24个积分与微分方程节点（曲线积分、场论、偏微分方程）
- Domain-3: 18个向量分析与优化节点（张量、变分法、最优控制）
- Domain-4: 10个概率统计节点（随机变量、常见分布、随机过程）
- Domain-5: 3个行业应用节点（AI、金融工程、物理建模）

---

### 2. 五大学域整合

**学域模式**将传统章节重组为五大数学学域：

1. **变化与逼近** 📈
   - 用离散逼近连续
   - 极限、导数、微分
   - 泰勒展开、数值方法

2. **结构与累积** 🔄
   - 从局部累积到整体
   - 积分、级数、微分方程
   - 曲线积分、曲面积分

3. **优化与决策** 🎯
   - 多元系统最优决策
   - 极值问题、约束优化
   - 变分法、最优控制

4. **不确定性处理** 🎲
   - 无穷与随机工具
   - 概率论、数理统计
   - 随机过程、贝叶斯推断

5. **真实问题建模** 🌐
   - 综合运用解决复杂问题
   - 工程应用、数据科学
   - AI、金融、物理建模
