# 🎯 快速参考卡片

**一页纸掌握所有功能**

---

## 🚀 启动 (3步)

```bash
cd knowledge-graph
python -m http.server 8000
# 打开 http://localhost:8000
```

---

## 🎮 基本操作

| 操作 | 方法 |
|------|------|
| **放大** | 点击 `+` 或滚轮向上 |
| **缩小** | 点击 `-` 或滚轮向下 |
| **平移** | 拖拽背景 |
| **重置** | 点击 `⟲` |
| **查看详情** | 点击节点 |
| **拖动节点** | 拖拽节点 |

---

## 🔍 筛选功能

| 筛选类型 | 位置 | 说明 |
|---------|------|------|
| **搜索** | 侧边栏顶部 | 输入关键词 |
| **学域** | 侧边栏中部 | 勾选学域 |
| **章节** | 侧边栏中部 | 选择章节 |
| **难度** | 侧边栏中部 | 调整滑块 |
| **清除** | 侧边栏底部 | 点击按钮 |

---

## 📖 节点详情

**打开**: 点击节点  
**关闭**: 点击 `×`

**包含内容**:
- 名称和描述
- 所属学域
- 难度等级
- 数学公式
- 前置知识
- 相关Skills

---

## 🛤️ 学习路径

**生成**: 详情面板 → "生成学习路径"  
**查看**: 图谱高亮显示  
**清除**: 点击 "清除路径"

**路径显示**:
- 绿色节点 = 路径节点
- 绿色连线 = 路径连接
- 数字标记 = 学习顺序

---

## 🎯 Skills系统

### 7大Skills

| Skill | 图标 | 类型 | 时长 |
|-------|------|------|------|
| 梯度可视化 | 🎨 | 可视化 | 45分钟 |
| 概念可视化 | 🎨 | 可视化 | 60分钟 |
| 推导动画 | 🎬 | 动画 | 40分钟 |
| H5P交互 | 🎮 | 交互 | 90分钟 |
| 极限与连续 | 📈 | 可视化 | 75分钟 |
| 导数与微分 | 📊 | 可视化 | 90分钟 |
| 积分概念 | 📐 | 可视化 | 105分钟 |

### 使用方法

1. 打开节点详情
2. 找到 "相关Skills"
3. 点击Skill按钮
4. 浏览4个标签页:
   - 理论
   - 可视化
   - 练习
   - 应用

## 🔗 跨学域视图

**启用**: 点击 `🔗` 按钮  
**显示**: 只显示跨学域连接  
**退出**: 再次点击 `🔗`

## 📱 移动端

| 操作 | 手势 |
|------|------|
| **平移** | 单指拖动 |
| **缩放** | 双指捏合 |
| **点击** | 单击节点 |

## 🎓 学习流程

```
1. 搜索/筛选 → 找到目标知识点
2. 点击节点 → 查看详情
3. 生成路径 → 规划学习顺序
4. 使用Skills → 深入学习
5. 做练习 → 巩固知识
```

---

## 🔧 常见问题

| 问题 | 解决方法 |
|------|---------|
| 加载不出来 | `Ctrl+F5` 硬刷新 |
| 节点太多 | 使用筛选功能 |
| 看不清 | 放大视图 |
| 找不到节点 | 使用搜索 |
| Skills打不开 | 刷新页面 |

## 📊 5大学域

| 学域 | 图标 | 核心概念 |
